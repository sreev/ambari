This package defines the entities (messages) for Ambari's REST interface. The primary two are Stack, which defines what needs to be deployed and how it should be configured, and ClusterDefinition, which binds a set of nodes to a given stack.

These entities are defined using JAXB and thus can be represented as either XML or JSON with a very simple mapping. Each of the top level entity classes in this package have the compact RelaxNG schemas for their type.

Any new classes that are added, must be manually inserted into org.apache.ambari.controller.rest.resources.ContextProvider.
